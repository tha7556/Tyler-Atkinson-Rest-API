Portfolio Website

Background

Prior to this, I had very little experience in web development or other front-end technologies. I started this in my spare time as a way to experiment with those technologies, and developed it to showcase my past projects.

Backend

In order to run some of my projects, which were already written in Java, I created a REST API to run them through the Spring framework. I had used a REST API before, but this was my first time developing my own. Luckily the Spring framework is very well documented and once it is set up, very easy to use. To host the backend I used Heroku's free plan. This means that the server sleeps after 30 minutes of inactivity, which can take some time to wake up. To avoid slowdowns while someone is trying to experiment with my projects, I added a REST request when the page is first loaded to begin waking up the server.

Frontend

I used Angular for the frontend, which I also had never worked in before this. Both Angular and Spring have excellent documentation, which is why I chose these two frameworks. As a beginner to frontend development, this was incredibly helpful. I also used a little bit of Bootstrap, mainly for the navigation bar. To host the frontend I used Google Firebase, also using the free plan.
